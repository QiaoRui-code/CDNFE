# CDNFE

Apply WGCNA to GEM
Generate undirected co-expression network----WGCNA.R
2.Directed Network Construction and filter the centre gene ---filter.m

The gene function annotation of each disease was obtained through GeneCards (http://www.genecards.org/ ).
